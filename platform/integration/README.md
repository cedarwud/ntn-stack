# 平台整合層

此目錄包含協助平台層與後端應用層整合的腳本和工具。

## 主要功能

-   提供與後端整合的庫和工具
-   實現 Open5GS 和 UERANSIM 管理的 API 接口
-   處理訊息傳遞和事件通知

## 目錄結構

```
integration/
├── api/               # API接口實現
├── adapters/          # 適配器和連接器
├── scripts/           # 整合腳本
└── utils/             # 輔助工具
```

## 使用方式

平台整合層作為平台層與應用層之間的橋樑，提供統一的 API 和工具，使應用層無需直接耦合到底層實現。
