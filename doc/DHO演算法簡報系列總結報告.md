### ✅ 公式詳細解釋
1. **MDP狀態空間設計**
   ```
   s[n] = {n, a^HO[n], a[n-1]}
   ```
   - n: 時間索引（軌道週期內位置）
   - a^HO[n]: 存取狀態向量
   - a[n-1]: 歷史動作記錄

2. **動作空間設計**
   ```
   a_j[n] = {a_0, a_1, a_2, ..., a_{K-1}}, Σa_k = 1
   ```
   - One-hot編碼確保互斥選擇
   - a_0: 不換手（智能退避）
   - a_k: 選擇目標衛星k

3. **獎勵函數設計**
   ```
   r[n] = -D[n] - ν·C[n]
   ```
   - D[n]: 正規化存取延遲
   - C[n]: 總碰撞率
   - ν: 應用相關權衡係數

4. **V-trace機制**
   ```
   v[n] = V(s[n]) + Σγ^(i-s)·Πc[j]·δ_i^V
   ```
   - 解決off-policy學習的策略滯後問題
   - 重要性採樣權重修正

5. **軌道預測基礎**
   ```
   q_i[m] = q_i[0] + τ·Σv_i[m'τ]
   ```
   - LEO衛星軌道確定性原理
   - 支援預測式決策的物理基礎
