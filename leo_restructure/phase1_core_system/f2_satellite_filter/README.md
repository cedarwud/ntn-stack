# F2: 衛星篩選引擎

智能篩選高品質候選衛星系統

## 核心功能
- 從~8,735顆篩選到~554顆候選 ⚠️ **預估值，待程式驗證**
- 地理相關性篩選 (NTPU觀測點優化)
- 星座特定評分系統
- 智能候選選擇演算法

## 篩選標準
### Starlink (目標：~350顆候選)
- 傾角權重: 30% (最佳53°)
- 高度權重: 25% (最佳550km)
- 相位分散: 20%
- 換手頻率: 15%
- 信號穩定: 10%

### OneWeb (目標：~204顆候選)
- 傾角權重: 25% (最佳87.4°)
- 高度權重: 25% (最佳1200km)
- 極地覆蓋: 20%
- 軌道形狀: 20%
- 相位分散: 10%
